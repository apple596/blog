速查：C++ 语法
====================

.. hint::

   本章是 C++ 基础语法的总结，适用于有其他编程语言经验的读者快速熟悉 C++ 语法。

C++ 基础语法包括：

* 语句以分号 ``;`` 结束。
* 注释：
  
  * 行内注释： ``//`` 将其同一行内右侧的内容标记为注释。
  * 多行注释： 包含在 ``/*`` 与 ``*/`` 内部的内容标记为注释。
    
    * 多行注释 **不支持嵌套** ，因此在调试中注释掉多行代码时，推荐用行内注释的方式注释每一行。 

* 控制语句：
  
  * if 语句：支持 ``else if`` 分支语句块。

* 变量类型
  
  * 字符（char）使用单引号，字符串（string）使用双引号。不可混用。
